# Landing page - The Odin Project

A responsive landing page built as part of The Odin Project (TOP) curriculum. improved to include full responsiveness across devices and a mobile-friendly navigation menu with a hamburger toggle.

## Overview

This project focuses on building a modern landing page layout using core web technologies. Implementation emphasized layout and styling fundamentals. Improvement enhances usability and accessibility on smaller screens.

## Live Preview

[Github Pages](https://divine-obiorah.github.io/landing-page--the-odin-project/)

## Features

- Responsive layout for desktop, tablet, and mobile screens
- Mobile navigation with hamburger menu toggle
- Clean and structured HTML markup
- Vanilla JavaScript for navigation interactivity

## Technologies Used

- HTML5
- CSS
- JavaScript
